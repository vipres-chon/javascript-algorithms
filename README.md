Collections of javascript algorithms.

## 前言
常见前端算法集锦以及思想。

## 学习目标
+ 数据结构：数组 链表 栈 队列 树 图
+ 算法：递归 DFS BFS 排序 二分法 双指针
+ 思想：分治 贪心 回溯 动态规划 分支界限
+ 技巧：位运算 双指针 滑动窗口

## 算法

### 算法时间复杂度量级
+ 常数阶O(1)
+ 对数阶O(logN)
+ 线性阶O(n)
+ 线性对数阶O(nlogN)
+ 平方阶O(n²)
+ 立方阶O(n³)
+ K次方阶O(n^k)
+ 指数阶(2^n)

### 算法空间复杂度量级
+ O(1)
+ O(n)
+ O(n²)
