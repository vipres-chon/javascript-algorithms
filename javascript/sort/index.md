## 技巧总结
### 联想记忆
   **选 冒 插 快 归 希 桶 基 堆 计**

   想象一个人，选择一个帽子作为武器，插入到另外一个人的身体，并且希望他快点归西，死去，抢了他的身上的宝贝，拿了一个桶，在桶的基部，将宝贝堆起来，并计数有多少宝贝。(杀人夺宝的故事)

### 算法规律
  + 选择排序 全是o(n2)
  + 冒 插完全一致 最好o(n) 最坏o(n2) 平均o(n2)
  + 快 归 希 堆 比较类似
  + 桶 基 计比较类似
